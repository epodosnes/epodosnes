# koei-check — 公営競技 開催チェック

中央競馬・地方競馬・ボートレース・オートレースの「今日の開催場」を1画面で確認できる静的サイト。
GitHub Pages でそのまま公開できる、ビルド不要（Node.js不要）の構成です。

## ファイル構成

```
koei-check/
├── index.html        # 画面本体
├── assets/
│   ├── style.css     # スタイル
│   └── main.js       # 日付選択と描画ロジック
├── data/
│   └── data.js       # 開催データ (手動更新)
└── README.md
```

## 使い方

- ローカル確認: `index.html` をブラウザで開くだけ（サーバー不要）
- 公開: このフォルダをGitHubリポジトリにpushし、GitHub Pagesを有効化

## データ更新の方法

`data/data.js` の `days` に日付キーで開催情報を追記し、`updatedAt` を更新して push する。

```js
"2026-07-12": {
  jra:  [ { venue: "福島", grade: "七夕賞(G3)", day: null } ],
  nar:  [ ... ],
  boat: [ { venue: "戸田", grade: "一般", day: "最終日" }, ... ],
  auto: [ ... ]
}
```

- カテゴリキー: `jra` (中央競馬) / `nar` (地方競馬) / `boat` (ボートレース) / `keirin` (競輪) / `auto` (オートレース)
- 日付キーが無い日、カテゴリキーが無いカテゴリは「データ未取得」表示
- カテゴリキーが空配列 `[]` なら「開催なし」表示（確認済みで開催が無い日に使う）
- `grade`: 一般 / G3 / G2 / G1 / SG / 重賞 など（SG・G1〜G3・重賞は画面上で赤く強調表示される）
- `day`: 開催日目（不明なら `null`）

更新はClaudeに「開催データを更新して」と依頼すればOK。
